
# Projeto de Teoria dos Grafos

Esse projeto foi realizado para a disciplina de Teoria dos Grafos orientada pela professora Laura Alves, o intuito desse projeto foi aplicar o conhecimento aprendido na disciplina em uma atividade feita para testar as nossas habilidades em relação ao assunto.

O projeto se consiste em duas partes, a parte 1 se trata de um desafio onde nós deveriamos criar um dataset dos bairros da cidade de Recife, e nesse dataset seria armazenado todas as conexões entre bairros. Baseado nessas informações foi nosso trabalho analisar esse dataset e tirar o máximo de informações possível dele.

A parte 2 da atividade nos deu a liberdade de escolher um dataset já feito, para que nele nós pudessemos testar os algoritmos que nós criamos (BFS, DFS, Dijkstra, Bellman-ford) e tentar analisar outro dataset, dessa vez procurando tirar mais conclusões em relação aos algoritmos usados.

# Como rodar o projeto

Demonstrando como rodar o projeto na sua própria máquina

Primeiro é necessario clonar o repósitorio na própria máquina:
```
git clone https://github.com/luankat0/projeto-grafos.git
```

Criando o ambiente virtual
```
python -m venv venv
```

Rodando o ambiente virtual
```
.\venv\Scripts\activate
```

Instalando as dependências do projeto
```
pip install -r requirements.txt
```

Navegando para a o diretório de scripts do projeto
```
cd .\src\
```

Rodando todos os arquivos necessários para o projeto
```
py .\cli.py
```

Após esses comandos, o projeto terá todos os arquivos necessários para a visualização do resultado
